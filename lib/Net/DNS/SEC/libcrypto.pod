=head1 NAME

Net::DNS::SEC::libcrypto - Perl interface to OpenSSL libcrypto


=head1 DESCRIPTION

Perl XS extension providing access to the OpenSSL libcrypto library
upon which the Net::DNS::SEC cryptographic components are built.
